CSS Selectors & Visual Rules

Ruleset Terms

Term Definition Example
Selector The beginning of the ruleset used to target the element that will be styled. .classname (to select a class called "classname"
Declaration Block The code between (and including) the curly braces {} that contains the CSS declaration(s) {color:blue;}
Declaration The group name for a property and value pair that applies a style to the selected element. color:blue; (includes the property and the value)
Property The first part of the declaration that signifies what visual characteristic of the element is to be modified. color: (in color:blue;)
Value The second part of the declaration that signifies the value of the property. blue; (in color:blue;)

Selectors - Selectors are used to target the specific HTML element(s) that will be styled by CSS.

Selector type Selective action Example
Universal Selector Selects all elements of any type using the " * " character. * {} selects all elements of the page.
Element Selector Selects a general HTML element by simply using the name of the element w/o angled brackets h1 {} selects all h1 headings.
Class Selector Selects HTML with matching defined class attributes using a period followed by the class name. Classes are meant to be used/reused over many elements. IDs override the styles of types and classes, thus should be used sparingly and only on elements that need to always appear the same. .tableheadings {} is used to select a "class="tableheadings" defined section of HTML code.
ID Selector Used to select a single specific element for unique styling; can only be used once per page. Uses " # " as a selector. #picture1 is used to select a specific picture subtitle called "picture1"
Attribute Selector Used to select attributes within an HTML element to eliminate the need for new code like class or id for elements w/ specific attributes. Uses the attribute name surrounded by brackets as the selector and can get increasingly specific by adding type or specific attribute values. img [src*='winter'] {} selects an img element with an attribute of src that contains the string "winter".
Pseudo-class Used to give elements a different style based on user interaction, site navigation, or position in the document. Pseudo-classes include hover, focus, visited, disabled, and active. A pseduo-class can be attached to any selector and is written as a " : " followed by a name p:hover {background-color: lime;} would change the background of the paragraph to lime color when a mouse hovers over it...like this paragraph does!

Visual Rules - Below are some examples of CSS visual rules

Rule Action Example
Font-family Changes the typeface of the text on your web page font-family: "Brush Script MT";
changes the font to THIS elegant font!
Font-size Changes the size of the text. font-size: 35px;
changes text size to 35 pixels
Font-weight Controls how bold or thin a text appears. Can be an integer (typical increments of 100) between 1 (lightest) and 1000 (boldest) OR words "bold", "bolder", "normal", "light", or "lighter" font-weight:1; OR font-weight: 1000;
Lightest versus Boldest
Text-align Aligns text respective to the element that holds it (parent element) with values including "left", "center", "right", and "justify" (spaces out text in order to align with the right and left side of the parent element text-align: center
Center Alignment
